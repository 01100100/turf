Module: turf/polygon

Generates a new GeoJSON Polygon feature, given an array of coordinates and list of properties.
Parameters:
Name Type Description
rings Array.<Array.<number>> an array of LinearRings
properties Object an optional properties object

var poly1 = turf.polygon([[[20.0,0.0],[101.0,0.0],[101.0,1.0],[100.0,1.0],[100.0,0.0]]])
var poly2 = turf.polygon([[[20.0,0.0],[101.0,0.0],[101.0,1.0],[100.0,1.0],[100.0,0.0]]],
