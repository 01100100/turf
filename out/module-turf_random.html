Module: turf/random

Generates random GeoJSON data, including Points and Polygons, for testing and experimentation.
Parameters:
Name Type Attributes Default Description
type String <optional>
 'point' type of features desired. Valid values are 'points' or 'polygons'
count Number <optional>
 1 how many geometries should be generated.
options Object relevant to the feature desired. Can include:
Properties
Name Type Attributes Default Description
bbox Array.<number> a bounding box inside of which geometries are placed. In the case of points, they are guaranteed to be within this bounds, while polygons have their centroid within it.
num_ertices Number <optional>
 10 the number of vertices added to polygon features.
max_radial_length Number <optional>
 10 the total number of decimal degrees longitude or latitude that a polygon can extent outwards to from its center.
